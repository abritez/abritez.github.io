Display

Elements are broken up into a few types, most notably block-level and inline.

Block Level Elements

Some of common characteristics for these elements include:

  • if no width is set it will stretch to fit the size of it's parent element
  • if not hieght is set it will expand to fit the size of the child elements it is containing
  • it will be place below the previous element, unless other CSS rules are applied to it or surrounding elements

Examples of block-level elements include:

  • <p>
  • <h1>...<h6>
  • <ul> and <ol>
  • <li>
  • <div>

In-line Elements

Some of common characteristics for these elements include:

  • will ignore height and width properties
  • will stay on the same line so it won't drop to the next line, unless the previous content was a block element.

Examples of inline elements include:

  • <a>
  • <b>...<strong>
  • <i> and <em>
  • <img>
  • <button>
  • <span>

Other diplay properties

Inline-Block

In the event that you want turn a block element into an inline element, but would like to retain the hight and width properties you could use 'display:inline-block'.

Display None

Setting your element's display value to none in CSS, will hide it from view in the browser. This is commonly used to hide and show elements from Javascript.
