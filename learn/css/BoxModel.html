CSS Box Model

Introduction

Surrounding every single element that shows up in your browser, no matter the complexity, is a simple box.

This box is comprise of four parts.

  1. Content: This is your element. Your element could be an image, text, video, absolutely anything.
  2. Padding: This is the inner space between the content and border.
  3. Border: Just like the names implies. This is the border around the box (not the content). This border is always square, but there are some stylying options which we will go over later in this section.
  4. Margin: The margin is the outer space of the content.

In the button below we highlight all 4 parts of the Box model. The text label in the middle is the content. The space between the word "button" and the purple outline is the padding. The purple outline is border, and the space on the outer part of the button is the margin.

The Code

Now it's time to check out some of the code.

All the margin & padding properties can have the following values:

  • auto - the browser calculates the margin
  • length - specifies a margin in px, pt, cm, etc.
  • % - specifies a margin in % of the width of the containing element
  • inherit - specifies that the margin should be inherited from the parent element

Padding

Adjust the padding for a specific side of the box 
button{
	padding-top: 10px;
	padding-right: 10px;
	padding-bottom: 10px;
	padding-left: 10px;
}
Adjust the padding for all sides of the box 
button{
	padding: 10px;
}
Shorthand for adjusting padding on all sides of a box. The order of measurement is clock-wise: top, right, bottom, and left. 
button{
	padding: 10px 15px 10px 15px;
}

Margin

Adjust the margin for a specific side of the box 
button{
	margin-top: 10px;
	margin-right: 10px;
	margin-bottom: 10px;
	margin-left: 10px;
}
Adjust the margin for all sides of the box 
button{
	margin: 10px;
}
Shorthand for adjusting margins on all sides of a box. The order of measurement is clock-wise: top, right, bottom, and left. 
button{
	margin: 10px 15px 10px 15px;
}

Border

Border includes a variety of additional CSS attributes. Below are a few, but the full list could be found in the MDN web docs and on the web. 
button{
	border-style: solid;
	border-width: 2;
	border-color: green;
}
