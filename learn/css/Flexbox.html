Flexbox

The Flexbox Layout module was created to allow a more optimal approach for handling different screen sizes and different display device. At it's core it is collection of CSS properties that are used to align content and distribute space.

Flex containers (parent) control the layout of child elements. That means properties such as "float" aren't necessary since the child is no longer in charge of adjusting it's positional state. That also means that elements inside of flexbox elements are direction-agnostic as opposed to the regular layouts (block which is vertically-based and inline which is horizontally-based).

Using the standard box model approach, the example below relies on the child to set the rules for where it will land in space.

Display Values

display:flex

As mentioned above, when positioning elements using flexbox layout, the container control much of the stying. Creating a flex container is a simple as adding "display:flex" on the CSS of that parent. All children will automatically start responding as flexbox items, disregarding CSS rules such as float and margin.

The flex container is acts as a block element. In the example below you will notice that the red outline streatch across the entire screen.

display:inline-flex

If you'd like your flex container to act more like an inline property, much like a SPAN, you have the option to use "display:inline-flex".

Properties

flex-direction

This property defines the axis in which all child elements will flow. This includes not only horizontal and vertical, but also reversed.

  • row: the items will align from left to right
  • row-reverse: the items will align from right to left
  • column: the items will align from top to bottom
  • column-reverse: the items will align from bottom to top

flex-wrap

By default all the item in a flex line will force themselves to fit in one line. If you'd like to make sure that items flow to the next line, then flex-wrap might do the trick.

Possible options include:

  • nowrap: keep all items on one line.
  • wrap: multi-line wrapping from top to bottom.
  • wrap-reverse: multi-line wrapping from bottom to top.

flex-flow: <‘flex-direction’> || <‘flex-wrap’>

This property is short-hand for both flex-direction and flex-wrap

.flex-container {
  flex-flow: row wrap;
}

justify-content

  • flex-start (default): items are packed toward the start line
  • flex-end: items are packed toward to end line
  • center: items are centered along the line
  • space-between: items are evenly distributed in the line; first item is on the start line, last item on the end line
  • space-around: items are evenly distributed in the line with equal space around them
  • space-evenly: items are distributed so that the spacing between any two adjacent alignment subjects, before the first alignment subject, and after the last alignment subject is the same

align-items

If you would like to only adjust a specific item inside of a container then use "align-self"

  • flex-start: cross-start margin edge of the items is placed on the cross-start line
  • flex-end: cross-end margin edge of the items is placed on the cross-end line
  • center: items are centered in the cross-axis
  • baseline: items are aligned such as their baselines align
  • stretch (default): stretch to fill the container (still respect min-width/max-width)

align-content

This property helps to specify formating rules when the container has extra space on the vertical-access, apposed to justify-content that looks at individual elements in the horizontal-axis.

  • flex-start: lines packed to the start of the container
  • flex-end: lines packed to the end of the container
  • center: lines packed to the center of the container
  • space-between: lines evenly distributed; the first line is at the start of the container while the last one is at the end
  • space-around: lines evenly distributed with equal space between them
  • stretch (default): lines stretch to take up the remaining space

Flex Grow

The flex-grow CSS property specifies the flex grow factor of a flex item. It specifies what amount of space inside the flex container the item should take up. The flex grow factor of a flex item is relative to the size of the other children in the flex-container.
