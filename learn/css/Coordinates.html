Coordinates

There are a variety of methods to position element on your screen. Sometimes you may want to position your element relative to another element, or the document

Window

This is the default position type of an element. With this positioning, your block level elements will stack one ontop of the next. Properties such as right, left, top, and bottom, are not available to static elements. That means if you'd need to override the static property default with a type that supports it.
